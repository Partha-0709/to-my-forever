TO MY FOREVER - DEPLOYMENT GUIDE

1. Create a GitHub account.
2. Click 'New Repository'.
3. Name it: to-my-forever
4. Upload all files from this folder.
5. Go to Settings > Pages.
6. Under Source, select 'Deploy from a branch'.
7. Choose 'main' branch and '/root'.
8. Save.
9. Wait a few minutes.
10. Your site will be available at:
https://yourusername.github.io/to-my-forever

Optional:
- Add your music file as love.mp3 in the project folder.
